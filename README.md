# CODSOFT Java Programming Internship Tasks

This repository contains all the tasks completed as part of the CODSOFT Java Programming Internship.

## Tasks Included

### Task 1: Number Game
A console-based number guessing game where the user tries to guess a randomly generated number within a limited number of attempts.

**Features:**
- Random number generation
- Multiple attempts
- Score tracking
- Play multiple rounds

**File:**
- NumberGame/NumberGame.java

---

### Task 2: Student Grade Calculator
A Java application that calculates total marks, average percentage, and grade based on marks entered for multiple subjects.

**Features:**
- Calculate total marks
- Calculate average percentage
- Grade calculation
- User-friendly output

**File:**
- StudentGradeCalculator/StudentGradeCalculator.java

---

### Task 3: ATM Interface
A Java-based ATM simulation that performs banking operations such as deposit, withdrawal, and balance inquiry.

**Features:**
- Deposit money
- Withdraw money
- Check balance
- Menu-driven interface

**File:**
- ATMInterface/ATMInterface.java

---

### Task 4: Currency Converter
A Java application that converts currency values using predefined exchange rates.

**Features:**
- Currency conversion
- Multiple currency options
- Accurate calculations
- Simple interface

**File:**
- CurrencyConverter/CurrencyConverter.java

---

## Technologies Used
- Java
