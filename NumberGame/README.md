# Number Game

## Description
A simple Java-based Number Guessing Game where the user has to guess a randomly generated number within a limited number of attempts.

## Features
- Random number generation
- User input handling using Scanner
- Limited attempts
- Hints for high and low guesses
- Displays total attempts taken

## Technologies Used
- Java
- VS Code

## How to Run
1. Compile the program:
   javac Numbergame.java

2. Run the program:
   java Numbergame
