# ATM Interface

## Description
A Java-based ATM simulation that allows users to perform basic banking operations such as checking balance, depositing money, and withdrawing money.

## Features
- Check account balance
- Deposit money
- Withdraw money
- Menu-driven interface
- Input validation

## Technologies Used
- Java
- VS Code

## How to Run
1. Compile the program:
   javac ATMInterface.java

2. Run the program:
   java ATMInterface
