# Currency Converter

## Description
A Java application that converts one currency into another using predefined exchange rates.

## Features
- Multiple currency conversions
- User-friendly menu
- Fast calculations
- Supports common currencies such as INR, USD, EUR, and GBP

## Technologies Used
- Java
- VS Code

## How to Run
1. Compile the program:
   javac CurrencyConverter.java

2. Run the program:
   java CurrencyConverter
